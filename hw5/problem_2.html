Задача 2
$\gdef\l{\lambda}$ $\gdef\f{\frac{1}{2} x^\top Q x + c^\top x}$ $\gdef\dom{\mathbb{R}^n}$ $\gdef\sxi{\sum_{i=1}^n\mu_i(x_i^2 - 1)}$

$ \min_{x \in \dom} \quad f(x) = \f \\ \text{ s. t.: } \quad -1 \le x_i \le 1, \\ \text{где Q - симметричная положительно определённая матрица}\\ $

Сведем задачу к следующей:
$$ \min_{x \in \dom} \quad f(x) = \f \\ \text{ s. t.: } \quad x_i^2 - 1 \le 0, \\ $$ $L(x, \mu) = \f + \sxi$

$ g(\mu) = \inf_{x} \{ \f + \sxi \} = \\ = \inf_{x} \{ \f + x^\top diag(\mu) x - \sum \mu_i \} = \\ = \inf_{x} \{ x^\top (Q + diag(\mu)) x + c^\top x - \sum \mu_i \}\\ $
$ Q + diag(\mu) \succeq 0, \text{ т.к. } \mu_i \ge 0 \rArr \inf_{x} \{ x^\top (Q + diag(\mu)) x \} = 0\\$ $ g(\mu) = \begin{cases} -\sum \mu_i, & \text{если } c = 0, \\ -\infty & \text{иначе} \\ \end{cases} \\ $

Двойственной задачей будет:
$ \max g(\mu) = - \sum \mu_i \\ \text{ s. t.: } \mu_i \ge 0, с = 0. \\ $ $\mu^* = 0, \> g(\mu^*) = 0$
Имеет место сильная двойственность, а значит $f(x^*) = 0$
