Задача 1
$\gdef\f{\frac{1}{2} \| x - c\|_2^2}$ $\gdef\h{a^\top x - b}$ $\gdef\hs{a^\top x^* - b}$ $\gdef\dom{\mathbb{R}^n}$ $\gdef\am{a^\top \mu^*}$

$ \min_{x \in \dom} \quad f(x) = \f \\ \text{ s. t.: } \quad a^\top x \leq b $

  • $\nabla f(x) = x -c$
    $\nabla^2 f(x) = I \succ 0 \rArr f(x) \text{ — выпуклая}$
  • $dom \> f = \dom \rArr \exist \bar{x} : a^\top \bar{x} < b$
  • $L(x, \mu) = \f + \mu(\h)$
1, 2) $\rArr$ выполнены условия ККТ, поэтому:
  • $ a^\top x^* \le b $
  • $ \mu^* \ge 0 $
  • $ \mu^* (\hs) = 0 $
  • $ \nabla L(x^*, \mu^*) = x^* - c + \mu^* a = 0 \rArr x^* = c - \mu^* a $
1, 4) $ \rArr a^\top (c - \mu^*a) \le b $
$ a^\top c - \mu^* a^\top a \le b $
$ \mu^* ||a||^2 \ge a^\top c - b $
  • Если $||a||^2 = 0, \text{то }\\ a = 0, \> x^* = c $
    $f(x^*) = 0, \\ b \ge 0$
  • $\gdef\mcalc{max \{0, \frac{a^\top c - b}{||a||^2} \}}$ $\gdef\mfrac{\frac{a^\top c - b}{||a||^2} }$ Если $||a||^2 \ne 0, \text{то }\\ \mu^* = \mcalc$
    $x^* = c - \mcalc a$
    При $\mfrac \le 0, \quad a^\top c \le b \> (т.к. \> ||a||^2 \ge 0), \quad x^* = c$.
    При $\mfrac > 0, \quad x^* = c - (\mfrac) a,$
    $a^\top (c - (\mfrac)a) \le b, \\$ $a^\top c - a^\top c - b \le b, \\$ $-b \le b \rArr b \ge 0$

Ответ:
$$ x^* = \begin{cases} c, & \text{если } a = 0, b \ge 0, \\ c - \mcalc a, & \text{если } a \ne 0, \\ \text{не существует}, & \text{если } a = 0, b < 0 \\ \end{cases} \\ $$
